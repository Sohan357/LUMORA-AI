# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Sohan357/LUMORA-AI.git
cd LUMORA-AI
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser.
